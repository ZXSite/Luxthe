LuxTHE is a modern, responsive Hugo theme with three-column layout, dark mode, and full-text search.

## Features

- Three-column responsive layout
- Dark mode with system preference detection
- Full-text search via FlexSearch
- i18n support (Chinese and English)
- Reading time estimates
- Category and tag taxonomies
- Archive page with yearly grouping
- Table of contents
- Responsive images
